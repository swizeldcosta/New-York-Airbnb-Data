# New York Airbnb Data Analysis

An in-depth analysis of New York Airbnb listings to study rental trends, pricing patterns, host activity, and customer engagement. The project is implemented using Jupyter Notebook for data exploration, cleaning, and visualization.

## 📊 Dataset Overview
**Dataset Name:** New York Airbnb Dataset

**Fields included:**
- `id` – Listing ID  
- `name` – Listing name  
- `host_id` – Host unique ID  
- `host_name` – Name of the host  
- `neighbourhood_group` – Grouped borough (e.g., Manhattan, Brooklyn)  
- `neighbourhood` – Specific neighbourhood  
- `latitude`, `longitude` – Location coordinates  
- `room_type` – Type of room (Entire home/apt, Private room, etc.)  
- `price` – Price per night  
- `minimum_nights` – Minimum nights required for booking  
- `number_of_reviews` – Total reviews received  
- `last_review` – Date of the last review  
- `reviews_per_month` – Average reviews per month  
- `calculated_host_listings_count` – Number of listings by the host  
- `availability_365` – Availability throughout the year  
- `number_of_reviews_ltm` – Reviews in the last 12 months  
- `license` – License information  
- `rating` – Guest rating  
- `bedrooms`, `beds`, `baths` – Property details  

## 🎯 Project Objectives
- Analyze distribution of Airbnb listings across New York boroughs.  
- Study pricing patterns and factors affecting rental costs.  
- Evaluate host activity and property availability.  
- Explore guest ratings, reviews, and occupancy trends.  

## 🛠️ Tools & Technologies
- **Jupyter Notebook**  
  - Data cleaning & preprocessing  
  - Exploratory data analysis (EDA)  
  - Data visualization (using Pandas, Matplotlib, Seaborn)  

## 📈 Key Insights (to be added after analysis)
- Popular neighbourhoods for Airbnb listings.  
- Room type and pricing trends.  
- Host listing patterns across the city.  
- Availability and seasonal trends in New York rentals.
